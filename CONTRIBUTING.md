# Contributing to Climate & Agriculture in Germany Project

Thank you for considering contributing to this project! We welcome ideas, feedback, and improvements from everyone in the group.

## How You Can Contribute

- Report bugs or suggest enhancements
- Improve code or add new functionality
- Contribute statistical models or analysis scripts
- Improve documentation or visualizations
- Help with data cleaning or organization
- Review pull requests or offer constructive feedback

## ️Setup Instructions

To contribute code:

1. **Fork the repository** (or clone it if you're already a collaborator):

    ```bash
    git clone https://gitup.uni-potsdam.de/khanal/group-5-name-tbd.git
    cd group-5-name-tbd
    ```

2. **Create a virtual environment** (recommended):

    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```

3. **Install dependencies**:

    ```bash
    pip install -r requirements.txt
    ```

4. **Create a new branch** for your changes:

    ```bash
    git checkout -b feat/your-feature-name
    ```

5. **Use Conventional Commits**:

    Please use [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) when writing commit messages, such as:

    - `feat(model): add lagged regression term`
    - `fix(data): handle missing temperature values`
    - `docs(readme): update setup instructions`

---

## Testing

Please ensure your contributions don’t break existing functionality. Add tests or comments if needed for clarity.
